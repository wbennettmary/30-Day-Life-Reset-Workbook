## 3. Export — US Letter PDF

1. Top-right → **Share** → **Download**
2. File type: **PDF Print**
3. Crop marks and bleed: **OFF** (digital download — no print bleed needed; if you turn this on for a future print version, save as a separate file)
4. Color profile: **RGB** (digital screens) — NOT CMYK
5. Flatten PDF: **ON** (prevents text reflow on the buyer's device)
6. Click **Download**
7. When the download finishes, save as:
   `30-day-life-reset/exports/Main_Workbook_US_Letter.pdf`

---

## 4. Export — A4 PDF

If you built the design at US Letter, duplicate first:

1. In Canva, with the US Letter design open → top menu → **Resize** (Pro feature)
2. Choose **A4** (210 × 297 mm)
3. Apply — Canva creates a copy of the design at A4 dimensions
4. Save the new design as: **`30DLR — A4`**
5. **Spot-check 3 pages** for text overflow:
   - Page 02 (Quick Start)
   - Page 25 (Week 3 Overview — usually the longest copy)
   - Page 53 (50 Tiny Habits — densest layout)
6. Fix any overflow by enlarging boxes or reducing font 1 pt where needed
7. Repeat the export flow from §3 — save as:
   `30-day-life-reset/exports/Main_Workbook_A4.pdf`
